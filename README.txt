Basic generic I/O replacement .dll for arcade games of a certain company.

---------------------------------------

To use, rename ttio.dll to whatever the original .dll is (ArrowheadUSBIO.dll, VisionUSBIO.dll, etc)
The default keybindings are for "Vision", but you can bind any of the 32 input pins of a TT I/O board by setting
keyN=0xwhatever
in the .ini file
